ขั้นตอนที่ 1 — เลือกฟีเจอร์ที่ต้องการใช้

เลือกได้มากกว่าหนึ่งอย่าง สามารถเปลี่ยนได้ภายหลังในหน้าตั้งค่า

ขั้นตอนที่ 2 — ตั้งค่าโปรไฟล์และเชื่อมต่อ API

ทุกช่องในขั้นตอนนี้ไม่บังคับทั้งหมด — ถ้าใช้แค่บันทึกรายรับ-รายจ่าย ข้ามไปขั้นตอนถัดไปได้เลย โดยไม่ต้องกรอก API ใดๆ

สมัครฟรีที่ openrouter.ai → Sign Up → API Keys → Create Key

สมัครฟรีที่ alpaca.markets → Create Account → Paper Trading → API Keys · ใช้ได้โดยไม่ต้องฝากเงิน

สมัครฟรีที่ financialmodelingprep.com → Register → Dashboard → API Key · ฟรี 250 req/วัน ไม่ต้องใส่บัตรเครดิต

ฟีเจอร์ที่เลือกไม่จำเป็นต้องใช้ API Key — กดถัดไปได้เลย

ขั้นตอนที่ 3 — สำรองข้อมูลด้วย GitHub Gist

ข้อมูลทั้งหมดจะถูก sync ไว้ที่ Gist ส่วนตัว เข้าถึงได้จากทุกอุปกรณ์

GitHub → Settings → Developer settings → Personal access tokens (Classic) → scope: gist
